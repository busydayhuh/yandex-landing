# Клуб четырех коней

Тестовый адаптивный лендинг по макету Figma.

## Стек

HTML, CSS, JavaScript без библиотек и фреймворков.

## Реализовано

- адаптивная верстка;
- Pixel Perfect по макету;
- оптимизация изображений в WebP;
- локальное подключение шрифтов;
- якорные ссылки на первом экране;
- бесконечная бегущая строка;
- карусель участников с автопрокруткой каждые 4 секунды;
- мобильная карусель этапов без автопрокрутки и зацикливания;
- базовая доступность и hover-состояния.

## Запуск

Открыть `index.html` в браузере.
